# md2word

一个用于将 Markdown 论文高质量转换为 Word（.docx）格式的开源工具，支持自定义模板与格式化宏，适合学术论文写作。

## 特性
- 支持 Markdown 转 Word
- 支持自定义 Word 模板
- 支持 Word 宏批量格式化
- 一键命令行操作

## 安装
```bash
pip install -r requirements.txt
```

## 使用
```bash
python -m md2word.cli input.md -o output.docx
```
